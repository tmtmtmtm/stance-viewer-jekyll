What is this?

This is a simple demo site for Stancer — a tool built on top of VoteIt to allow you to display how politicians and political groups voted on Issues like banning fox hunting, reducing spending on welfare benefits, allowing same-sex marriage, or introducing a bankers' bonus tax.

Although based on real data from the UK, it's only a demo, so don't rely on it for anything!

The tools used to build this are designed to make it easy to build similar sites all around the world. So if you'd like to see something like this for your country, contact me:
